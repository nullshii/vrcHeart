### Fake heart beat monitor for VRChat


## How to use

1. Download app from [releases page](https://github.com/nullshii/vrcHeart/releases)
2. Launch app (it will create `settings.json` file)
3. Open `settings.json`
4. Modify text however you want
5. Relaunch to load new settings
6. Have fun :D

## Settings guide
| Parameter     | Description                                                                                                        | Available values                         | Default   |
| ------------- | ------------------------------------------------------------------------------------------------------------------ | ---------------------------------------- | --------- |
| Address       | VRChat osc address                                                                                                 | Any IP Address                           | 127.0.0.1 |
| Port          | VRChat osc port                                                                                                    | from 0 to 65535                          | 9000      |
| SendFrequency | Chatbox update rate (in milliseconds) (not recommended too low frequency because you gonna get temporary chat ban) | from 0 to 2147483647                     | 1500      |
| MinRate       | Minimum number (bpm)                                                                                               | -2147483648                              | 70        |
| MaxRate       | Maximum number (bpm)                                                                                               | 2147483647                               | 190       |
| StartRate     | Start number (bpm)                                                                                                 | from 0 to 2147483646                     | 120       |
| LeftEmoji     | Emoji on left side of number                                                                                       | Any text                                 | ♥         |
| RightEmoji    | Emoji on right side of number                                                                                      | Any text                                 | ♥         |
| TextAbove     | Text above bpm                                                                                                     | Any text                                 |           |
| TextBelow     | Text below bpm                                                                                                     | Any text                                 |           |
| NumberType    | BPM Type                                                                                                           | 0 - Random, 1 - increment, 2 - decrement | 0         |
| SaveLastRate  | Save last BPM to next start rate (useful for saving increment or decrement rates)                                  | true, false                              | false     |
